# 本地开发指南（无需 Docker）

如果 Docker 镜像拉取遇到网络问题，可以使用本地开发方式运行项目。

## 前置要求

- Python 3.11+ 
- Node.js 18+
- npm 或 yarn

## 后端本地运行

### 1. 进入后端目录
```powershell
cd D:\selfCode\query-generator\backend
```

### 2. 创建虚拟环境（推荐）
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

如果遇到执行策略错误，运行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 3. 安装依赖
```powershell
pip install -r requirements.txt
```

如果 pip 安装慢，可以使用国内镜像：
```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 4. 运行后端
```powershell
python app.py
```

后端将在 `http://localhost:5000` 运行

## 前端本地运行

### 1. 打开新的终端，进入前端目录
```powershell
cd D:\selfCode\query-generator\frontend
```

### 2. 安装依赖
```powershell
npm install
```

如果 npm 安装慢，可以使用国内镜像：
```powershell
npm install --registry=https://registry.npmmirror.com
```

### 3. 运行前端
```powershell
npm run dev
```

前端将在 `http://localhost:5173` 运行

## 访问应用

打开浏览器访问：`http://localhost:5173`

前端会自动代理 API 请求到后端的 `http://localhost:5000`

## 停止服务

在对应的终端窗口按 `Ctrl + C` 停止服务
